
## 项目初始化 ##
```
npm init
```

## 依赖模块安装

```
npm install 

```

## 结构目录
```
* db                数据库存储目录
* models            数据库模型文件目录
* node_modules      node第三方模块目录
* public            公共文件目录（CSS、JS、image...）
* routers           路由文件目录
* schemas           数据库结构文件（schemas）目录
* views             模板视图文件目录
* app.js            应用（启动）入口文件
* package.json
* gulpfile.js       gulp
* wechat            微信文件
```



## 数据库

```
mongodb
```


## 启动app.js 
```
node app.js
```
